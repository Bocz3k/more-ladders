# More Ladders
### Mod that adds more types of ladders from wood types and other things
Currently Supporting 12 ladders:
- Acacia ladder
- Bamboo ladder
- Birch ladder
- Chain ladder
- Cherry ladder
- Crimson ladder
- Dark oak ladder
- Iron ladder
- Jungle ladder
- Mangrove ladder
- Spruce ladder
- Warped ladder
- Gold ladder
